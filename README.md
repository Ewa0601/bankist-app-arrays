## 🏦 Bankist App (Arrays)

A simple banking application that simulates core banking features such as logging in, transferring money, requesting loans, and viewing transaction history. The project focuses on working with JavaScript arrays and modern array methods.

---

## 🚀 Features

- User login system (with demo accounts)
- Display of transaction history
- Money transfers between accounts
- Loan request functionality
- Account balance calculation
- Sorting transactions
- Auto logout timer for security

---

## 🛠️ Technologies

- HTML
- CSS
- JavaScript (ES6+)

---

## 🧠 Key Concepts

This project focuses on practicing:

- Array methods (map, filter, reduce, find, sort)
- DOM manipulation
- Event handling
- Working with objects and data structures
- Timers (`setInterval`) and app state management

---

## 🎮 How It Works

1. Log in using a demo account (username & PIN)
2. View your transactions and current balance
3. Transfer money to another user
4. Request a loan (if conditions are met)
5. Sort transactions or log out automatically after inactivity

---

## 📚 Purpose

This project was created as a learning exercise to practice advanced JavaScript concepts, especially working with arrays and building interactive applications without a backend.
